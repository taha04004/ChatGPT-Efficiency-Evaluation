Project Overview

This project evaluates ChatGPT‚Äôs performance across multiple academic domains by measuring accuracy, response time, and reasoning consistency. Using MongoDB, Node.js, WebSockets, and the ChatGPT API, the system automates question retrieval, response analysis, and real-time visualization.

Evaluation Setup

The backend fetches questions, sends them to ChatGPT, logs latency, and matches accuracy keywords.

ChatGPT Evaluation Method

The system evaluates ChatGPT by sending each question to the ChatGPT API, recording its generated answer, calculating response time, and verifying correctness against expected keywords stored in MongoDB.
